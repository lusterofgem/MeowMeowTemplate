All elements in html got three kind of spacing attributes: margin, border and padding.

margin:

Margin is the inherent space between the element and other element.

Think of the element is very hate crowding, it doesn't want to touch with other element, than it might have a large margin.


border:

Border is the outline of the element.

But it is not necessary to be a thin line, it can be very thick, even change the color is feasible


padding:

Padding is the inner space between the element boundaries and the contant

You can think of the contant want to live in a big house, that it have many space around.

The three attributes, they can be specified by add the css style.




Note: We can also specify a specific direction like margin-top or padding-bottom.
