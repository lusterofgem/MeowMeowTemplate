Table is combine by four tags <table>, <tr>, <td> and <th>

<table> should contain the whole table, it the rule.

<tr> represent a single row, it usually contains more than one td

<td> represent a single column, contain one value.

<th> is just like <td>, but it's the table heading, usually bold.

Table is very useful when we compair the pros and cons of different things.

There are one important thing about table is they don't naturally bring up with border, you need to specify in css yourself.
