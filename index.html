This is a simple website to introduce and practice html.

A website is combine with three types of elements: html, css and javascript.

Html is combine with tags, there are many tags to represent the html structure.

They are just like annotations, choose sensible tag is helpful for other to understand the structure of your html file.

The three required tag you must remember is: <html>, <head> and <body>.

Usually you will put your import things like css or javascript in <head>, and put the contents in <body>.
